Scrabble

Jeu de Scrabble en Python réalisé dans le cadre d’un projet en premiere.

Description

Ce projet est une implémentation du jeu Scrabble en Python.
Il contient le code du jeu, les données nécessaires (lettres, mots, etc.) ainsi que des images utilisées pour l’affichage.

Lancer le jeu

Avoir Python 3 installé

Cloner le dépôt :

git clone https://github.com/MilanM-V/Scrabble.git
cd Scrabble


Lancer le jeu :

python jeu.py

Structure du projet
Scrabble/
├── donnees/   # Données du jeu
├── images/    # Images
├── site/      # Interface web (si utilisée)
├── jeu.py     # Fichier principal
└── README.md

Technologies utilisées

Python
