Expliquation du Code

Le code crée plusieur page Tkinter chacune par le biais de fonction sauf le page de connexion qui est la pge principale au qu'elle toute les page sont relier grace à TopLevel.
Il y a d'autres fonction qui permette par exemple de recommencer, de retourner a l'accueil, de changer de mode de jeu, d'initialiser les zone de drag et drop, ou de reinitialiser le jeu.
Il y aussi une class qui créer/permet le system de drag and drop et enregistre les lettres choisies dans la variable roster pour permettre la vérification du mots dans le dictionnaires

Pistes d'amélioration

Comme piste d'amelioration nous voyons:

- De pouvoir mettre les lettres dans l'ordre que l'ons veut dans la page de création de mot.
- Un meilleur visuel.
- Une traduction des texte en fonction de lalangue choisies.
- Une optimisation du code pour ameliorer ça vitesse.
- Et un meilleur visuel du chrono sur la page de création de mot lorsque qu'il passe en dessous de 10sec.
